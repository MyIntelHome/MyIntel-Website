# Photos

Drop image files here and they appear on the site automatically (no code change).
Until a file exists, the site shows a tasteful branded placeholder in its place.

## Where each photo appears

| Filename | Where it shows | Suggested shot |
| --- | --- | --- |
| `home-assessment.jpg` | Homepage → "Home Safety Assessment" section | A specialist and an older adult reviewing the home together; warm, natural light. Landscape, ~4:3. |

## How to add one

1. Save your image with the exact filename from the table into this `public/photos/` folder.
2. Use a landscape JPG, ideally ~1600px wide, compressed (aim under ~400 KB).
3. Commit and push — Vercel redeploys and the photo replaces the placeholder.

## Where to get rights-cleared stock photos (free for commercial use)

- Unsplash — https://unsplash.com (search "senior home care", "elderly couple home", "caregiver senior")
- Pexels — https://www.pexels.com
- Both are free for commercial use with no attribution required. Download, rename to the filename above, and drop it here.

Or paste an image link to Austin's assistant and it can add it for you.
